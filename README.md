# Enterprise RAG Chatbot

## Project Overview
This project builds a Retrieval-Augmented Generation (RAG) chatbot for querying legal contract PDFs.

## Features
- PDF ingestion
- Document chunking
- OpenAI embeddings
- Pinecone vector database
- Semantic search
- AI-powered question answering

## Technologies Used
- Python
- LangChain
- OpenAI
- Pinecone
- Google Colab
- FastAPI

## Workflow
1. Upload PDFs
2. Chunk documents
3. Generate embeddings
4. Store vectors in Pinecone
5. Query chatbot
